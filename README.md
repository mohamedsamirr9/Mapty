# Mapty
This application renders workouts based on the user's location.

## Features
<ul>
<li>A workout that could be chosen:</li>
  <ul>
  <li>Running</li>
  <li>Cycling</li>
  </ul>
<li>Editing workouts</li>
<li>Deleting workouts</li>
<li>Deleting all workouts</li>
</ul>
